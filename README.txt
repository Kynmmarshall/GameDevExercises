GameDevExercises - README
========================

Files:
------
exercise1.py            - Game loop with delta time
exercise2.pdf           - ECS design (see your PDF submission)
exercise3.py            - Input manager with observer pattern
exercise4.py            - Spatial grid collision detection
exercise5.py            - Stack-based game state machine
exercise6.py            - State-based sprite animation system (uses Assets/ for images)
exercise7.py            - Parallax scrolling background
exercise8.py            - Generic particle system (dust, explosion)

Assets/                 - Folder containing sprite images for animations (used in exercise6.py)
  idle/                 - Idle animation frames
  walking/              - Walking animation frames
  attacking/            - Attacking animation frames
  jump/                 - Jump animation frames

How to Run:
-----------
- All exercises require Python 3 and pygame installed.
- Run any exercise with:  python exerciseX.py  (replace X with the exercise number)
- For exercise6.py, make sure the Assets/ folder is present and contains the required images.
- For exercise2.pdf, open with a PDF viewer.

Special Instructions:
---------------------
- If you add new animation frames or assets, update the Assets/ folder accordingly.
- Some exercises (e.g., exercise4.py, exercise7.py, exercise8.py) can be visually enhanced by replacing placeholder graphics with your own images.
- If you encounter issues with pygame window not opening, ensure your Python environment is set up correctly and pygame is installed.
